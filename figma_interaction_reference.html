📱 Figma プロトタイプ / UI インタラクション種類一覧

スマートフォンアプリ・Webサービス制作における代表的なインタラクションをカテゴリ別に整理したリファレンス。

1. タッチ / ジェスチャーインタラクション

種類 説明 使用例
タップ 画面上の要素を1回押す ボタン、リンク
ダブルタップ 素早く2回押す 画像ズーム、いいね
長押し 一定時間押し続ける コンテキストメニュー
スワイプ 指を滑らせて移動 カルーセル、メニュー表示
ピンチイン / アウト 2本指で縮小・拡大 マップ、画像拡大
ドラッグ & ドロップ 要素を掴んで移動 並び替え、ファイル移動

2. マウス / ポインターインタラクション

種類 説明 使用例
ホバー ポインターを要素上に乗せる ツールチップ表示
クリック マウスボタンを押す ボタン、リンク
ダブルクリック 素早く2回クリック 編集開始
スクロール ページを上下移動 無限スクロール

3. ビジュアル・トランジション

種類 説明 使用例
フェードイン / アウト 徐々に表示・非表示 モーダル表示
スライドイン / アウト 横・縦に移動しながら表示 サイドメニュー
スケールアニメーション 拡大縮小の動き ボタン押下エフェクト
ローディング表示 処理中を視覚化 スピナー、進捗バー

4. フィードバックインタラクション

種類 説明 使用例
サウンド 音で操作結果を伝える 送信音、通知音
バイブレーション 振動によるフィードバック スマホタップ反応
ステータス変化 状態の視覚変化 ON/OFF切替
トースト通知 一時的な通知表示 保存完了通知

5. ナビゲーション・トランジション

種類 説明 使用例
ページ遷移アニメーション 画面遷移時の動き スマホアプリ遷移
タブ切り替え 内容を切り替えて表示 ニュースカテゴリ
モーダル / ポップアップ 一時的なオーバーレイ表示 確認ダイアログ

6. マイクロインタラクション

種類 説明 使用例
トグルアニメーション ON/OFF切替の細かな動き 設定画面
いいねアニメーション 操作に感情表現を加える SNSのハート演出
入力フィードバック リアルタイム検証 パスワード強度表示
